# توصيف مشروع منصة "التفاعل التعليمي" (Realtime Quiz App)

هذا التوصيف شامل لجميع المييزات التقنية والوظيفية للمشروع لاستخدامه في طلب اقتراحات تطويرية.

## 1. نظرة عامة (Overview)

منصة ويب تفاعلية تتيح للمعلمين إنشاء جلسات (Sessions) وطرح أسئلة (MCQ أو مقالية) على الطلاب بشكل فوري داخل الفصل. الطلاب ينضمون للجلسة ويجيبون من هواتفهم، وتظهر النتائج فوراً على شاشة المعلم (Projector) بدون الحاجة لتحديث الصفحة.

## 2. التقنيات المستخدمة (Tech Stack)

- **Frontend:** React + TypeScript + Vite
- **Styling:** Tailwind CSS + CSS Modules
- **Animations:** Framer Motion (للانتقالات السلسة والعناصر التفاعلية)
- **Backend & Database:** Supabase (PostgreSQL)
- **Realtime:** Supabase Realtime (للمزامنة الفورية للإجابات وحالة الأسئلة)
- **Authentication:** Supabase Auth (للمعلمين فقط)
- **Routing:** React Router DOM

## 3. المميزات الوظيفية (Key Features)

### أ) للمعلم (Teacher)

1.  **الصفحة الرئيسية (Landing Page):** واجهة اختيار بين "معلم" أو "طالب" بتصميم عصري.
2.  **لوحة التحكم (Dashboard):**
    - عرض قائمة الجلسات السابقة (مع حالتها: نشطة/غير نشطة).
    - إمكانية إنشاء جلسة جديدة (يتم توليد كود سداسي عشوائي `123456`).
    - تفعيل/إيقاف الجلسات بضغطة زر.
    - حماية البيانات: المعلم يرى جلساته فقط.
3.  **إدارة الجلسة (Session View):**
    - **إنشاء الأسئلة:**
      - **اختيار من متعدد (MCQ):** دعم اختيار إجابة واحدة أو إجابات متعددة (Multi-select).
      - **سؤال مقالي (Essay):** يكتب الطالب نصاً حراً.
    - **التحكم في الأسئلة:** زر "تفعيل" لإرسال السؤال للطلاب فوراً، وزر "التالي/السابق" للتنقل السريع.
    - **عرض النتائج (Live Results):**
      - **MCQ:** رسوم بيانية متحركة (Animated Bars) تظهر نسب الإجابات وعددها فوراً.
      - **Excel-like Grid:** (للأسئلة المقالية) بطاقات تظهر إجابات الطلاب فور وصولها، مرتبة من الأحدث للأقدم.
    - **وضع العرض (Projector Mode):** زر لتكبير السؤال والنتائج لملء الشاشة، مع إخفاء القوائم الجانبية للتركيز، مناسب للعرض على السبورة الذكية (Data Show).
    - **QR Code:** زر لإظهار كود QR كبير للطلاب لسهولة الانضمام.

### ب) للطالب (Student)

1.  **الانضمام (Join):**
    - إدخال "الكود" و "الاسم" فقط (بدون تسجيل دخول).
    - يمكن الانضمام عبر مسح QR Code مباشرة.
2.  **الانتظار (Waiting Room):** شاشة انتظار متفاعلة (Pulse Animation) تخبر الطالب بانتظار تفعيل المعلم للسؤال.
3.  **واجهة الإجابة:**
    - تظهر الأسئلة فور تفعيلها من المعلم (Realtime) دون تحديث الصفحة.
    - تصميم متجاوب (Mobile-First) كأنها تطبيق هاتف.
    - دعم اختيار أكثر من إجابة (Checkboxes) أو إجابة واحدة (Radio Buttons).

## 4. تجربة المستخدم والتصميم (UI/UX)

- **Dark Mode:** الموقع بالكامل يدعم الوضع الليلي (Dark Theme Only) بألوان عصرية (Deep Blue/Purple).
- **اللغة:** واجهة عربية بالكامل (RTL Support).
- **التفاعلية (Optimistic UI):** النتائج تظهر فوراً على شاشة المعلم بمجرد ضغط الطالب "إرسال"، مع معالجة ذكية للكاش لضمان السرعة.
- **Visuals:** استخدام أيقونات (Lucide React) وتأثيرات زجاجية (Glassmorphism).

## 5. الأمان (Security)

- **RLS (Row Level Security):**
  - المعلم يرى ويعدل بياناته فقط.
  - الطلاب (Anonymous) يمكنهم فقط إرسال الإجابات وقراءة الأسئلة النشطة فقط.
- **Session Isolation:** تم حل مشكلة تداخل الجلسات بحيث لا يرى المعلم سوى طلابه وجلساته.
